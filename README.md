# python_data_analysis_project

# Basic DNA Analysis

Python project demonstrating data handling, cleaning, and visualisation skills through DNA sequence analysis.

---

## What it does

- Reads DNA sequences from a FASTA file.
- Adds a user-provided sequence from NCBI.
- Cleans data: removes duplicates and sequences with errors.
- Calculates basic DNA properties: nucleotide counts (A, C, G, T) and GC content.
- Visualises data with charts:  
  - GC content per sequence

---

## Skills Highlighted

- Python programming (functions, loops, error handling)  
- Data structures: lists, dictionaries, sets  
- API usage: NCBI Entrez via Biopython  
- Data analysis & visualisation: Pandas, Matplotlib, Seaborn  

---

## How to run

1. Install libraries:
```bash
pip install biopython pandas matplotlib seaborn
```

Place sekwencje.txt in the project folder.


2. Run the script:

``` python Basic_dna_analysis.py ```
